## Internationalization (I18N)

This project supports multiple languages. The default language is English (`en`), and Spanish (`es`) is also supported.

### Adding a New Language

1.  **Update Settings**:
    Open `src/configuration/settings/base.py` and add the new language code and name to the `LANGUAGES` list.

    ```python
    LANGUAGES = [
        ('en', 'English'),
        ('es', 'Spanish'),
        ('fr', 'French'),  # New language
    ]
    ```

2.  **Create Translation Files**:
    Run the following command from the `src` directory to create the message file for the new language:

    ```bash
    django-admin makemessages -l fr
    ```

    This will create a `django.po` file in `src/locale/fr/LC_MESSAGES/`.

3.  **Translate**:
    Open the `django.po` file and add the translations for the `msgid` strings in the `msgstr` fields.

    ```po
    msgid "Hello"
    msgstr "Bonjour"
    ```

4.  **Compile Messages**:
    Run the following command to compile the translation files:

    ```bash
    django-admin compilemessages
    ```

### Updating Translations

If you change any translatable strings in the code (marked with `gettext`, `_`, or `{% trans %}`), run `makemessages` again to update the `.po` files, then translate the new strings and run `compilemessages`.
